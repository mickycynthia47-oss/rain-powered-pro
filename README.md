## 🎬 Splash Screen Experience

When you first visit the app, you'll see a **premium blue-themed splash screen**:

- 🌧️ **Animated rain drops** falling gracefully
- 💙 **Deep blue gradient background** (#0D47A1 → #1976D2 → #42A5F5)
- 🤖 + ⚡ + 📧 + 📊 **animated floating icons**
- 🎯 **"CLICK OK TO PROCEED" button** with pulsing animation
- ✨ **Fade-in transition** to main app

Once you click OK, the splash screen saves to `localStorage` and never appears again on that browser.

## 🧠 Prompt Engineering

### Email Generator Prompt
```
Write a [tone] email to [recipient] about [context].
Include: subject line, professional greeting, bullet points if helpful,
call-to-action, and professional closing. Keep under 200 words.
```

### Meeting Summarizer Prompt
```
Extract from meeting notes:
1. 3-5 key points (🔑 emoji)
2. Decisions made (✅ emoji)
3. Action items with owners (📋 emoji)
4. Deadlines (⏰ emoji)
Format as markdown. If info missing, say "Not specified."
```

[Additional prompts for Task Planner, Research Assistant, Chatbot available in the app's Settings → Prompt Library]
